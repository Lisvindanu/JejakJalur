Destinasi

GET /destinasi

Daftar destinasi dengan filter & pagination. Render: Destinasi/Indeks

Query params (filter)

ParamKeterangan
kata_kuncifull-text search nama
kota_idUUID kota
stasiun_idUUID stasiun
kategoriWisata | Kuliner | UMKM

Inertia props

{
  destinasi: PaginatedData<Destinasi>,  // tanpa ulasan
  semuaKota: Kota[],                      // include stasiun[]
  filter: { kata_kunci, kota_id, stasiun_id, kategori }
}
GET /destinasi/{id}

Detail destinasi. Render: Destinasi/Detail

Inertia props

{
  destinasi: Destinasi  // include stasiun.kota + ulasan[].user + foto_url
}
Gunakan foto_url (bukan foto) untuk tag <img>. foto_url adalah public URL R2.

Ulasan

POST /destinasi/{destinasi}/ulasan auth

Tambah ulasan baru.

FieldValidasi
ratingrequired, integer, 1–5
judulnullable, max:100
kontenrequired, max:1000
Field-nya rating dan konten (bukan nilai/komentar).
PATCH /destinasi/{destinasi}/ulasan/{ulasan} auth + owner

Edit ulasan milik sendiri. Field sama seperti POST.

DELETE /destinasi/{destinasi}/ulasan/{ulasan} auth + owner

Hapus ulasan milik sendiri.
